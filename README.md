# 📊 Sales Performance Dashboard

## 🎯 Project Overview
This project focuses on analyzing sales data to identify trends, seasonal patterns, and the performance of different products. An interactive dashboard was created using Microsoft Excel to provide meaningful business insights.

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

---

## 📈 Dashboard Features
✔ Total Sales Analysis  
✔ Total Profit Analysis  
✔ Monthly Sales Trend  
✔ Top Products by Sales  
✔ Sales by Region  
✔ Sales by Category  
✔ Profit by Category  
✔ Interactive Filters (Slicers)

---

## 📂 Dataset Information
The dataset contains:
- Order Details
- Sales Data
- Profit Data
- Product Categories
- Regional Data
- Order Dates

---

## 🔍 Key Insights
- Identified monthly sales trends and growth patterns.
- Compared sales performance across different regions.
- Analyzed category-wise profit contribution.
- Identified top-performing products.
- Evaluated the impact of discounts on profitability.
